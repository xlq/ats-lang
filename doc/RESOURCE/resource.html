Resources for ATS/Anairiats Users

This website contains links to various resources for ATS/Anairiats users.

ATS/Anairiats User's Guide  (pdf) (ps)

Emacs Mode  A mode for editing ATS source code in Emacs is available here.

Contributed Code  The primary purpose of contributed code is to facilitate programming in ATS. Also, this is a place for certain code to be properly tested before it can be moved into the library of ATS. Please contribute!

  • Linear Stack (dats) A linear stack implementation based on lists.
  • Linear Queue (dats) A linear queue implementation based on lists.
  • Functional Array (dats) A functional array implementation based on Braun trees.
  • Functional Set (dats) A functional set implementation based on AVL trees.
  • Functional Map (dats) A functional association map implementation based on AVL trees.
  • Functional Red-Black Tree (dats) A functional red-black tree implementation.
  • Hashtable (dats) A hashtable implementation where buckets are represented as linked lists.
